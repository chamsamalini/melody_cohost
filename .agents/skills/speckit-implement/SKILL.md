# Spec Kit Implement

Read and follow `.specify/templates/commands/implement.md`.

When the command references `scripts/powershell/check-prerequisites.ps1`, run `.specify/scripts/powershell/check-prerequisites.ps1` from the repository root.

Implementation must follow:

- `.specify/memory/constitution.md`
- `.specify/context-augmentation.md`
- Active `specs/<feature>/spec.md`
- Active `specs/<feature>/plan.md`
- Active `specs/<feature>/tasks.md`
